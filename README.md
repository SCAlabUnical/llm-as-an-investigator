# LLM-as-an-Investigator

**Evidence-First Reasoning for Robust Interactive Problem Diagnosis**

This repository contains the code, prompts, and dataset used in the paper:

**LLM-as-an-Investigator: Evidence-First Reasoning for Robust Interactive Problem Diagnosis**

The project studies how Large Language Models can be used as interactive diagnostic assistants. Instead of giving an immediate answer from an incomplete problem description, the proposed approach asks clarification questions, considers multiple possible causes, and reaches a final diagnosis only after collecting enough evidence.

The main goal is to make LLM-based problem diagnosis more reliable and more robust to misleading user assumptions.
